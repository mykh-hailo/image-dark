## Screenshots (light / dark)

GitHub supports automatic image switching using `prefers-color-scheme` via `<picture>`.
Put your images in `assets/` (or update the paths below).

  <source media="(prefers-color-scheme: dark)" srcset="dark.jpg">
  <source media="(prefers-color-scheme: light)" srcset="light.jpg">
  <img alt="Project screenshot" src="light.jpg">
</picture>
